# DIM

This repository contains the core code to implement the idea of  our paper:

[Discover and Mitigate Multiple Biased Subgroups in Image Classifiers](https://arxiv.org/abs/2403.12777) (CVPR 2024)


## Requirements

+ python >= 3.6.5
+ pytorch == 1.7.x
+ numpy >= 1.15.4
+ clip-retrieval >= 2.0
